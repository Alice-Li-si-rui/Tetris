# 俄罗斯方块 (Tetris)

一个用Python和Pygame实现的经典俄罗斯方块游戏。

## 项目结构

```
Tetris/
├── README.md          # 项目说明文件
├── main.py           # 游戏主文件
├── tetris.py         # 游戏核心逻辑
├── constants.py      # 游戏常量定义
├── requirements.txt  # 依赖包列表
└── assets/           # 资源文件夹（可选）
    └── sounds/       # 音效文件（可选）
```

## 功能特性

- 经典俄罗斯方块玩法
- 7种标准方块形状
- 实时得分系统
- 级别提升机制（速度随级别增加）
- 下一个方块预览
- 键盘控制（左右移动、旋转、加速下落）
- 游戏暂停/继续功能
- 游戏结束检测

## 安装与运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行游戏

```bash
python main.py
```

## 控制说明

- **左右方向键**：移动方块
- **上方向键**：旋转方块
- **下方向键**：加速下落
- **空格键**：快速下落
- **P键**：暂停/继续游戏
- **Q键**：退出游戏

## 游戏规则

1. 方块从顶部随机生成并下落
2. 玩家可以移动和旋转方块
3. 当方块落地或碰到其他方块时停止
4. 当一行被方块填满时，该行消除并得分
5. 随着得分增加，游戏级别提升，下落速度加快
6. 当方块堆叠到顶部时，游戏结束

## 技术实现

- **语言**：Python 3
- **图形库**：Pygame
- **游戏循环**：使用Pygame的事件循环
- **碰撞检测**：自定义算法
- **渲染**：Pygame的绘图API

## 开发环境

- Python 3.7+
- Pygame 2.0+
